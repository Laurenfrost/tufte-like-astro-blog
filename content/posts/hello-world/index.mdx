## 中英文混排

Good typography is invisible.<Sidenote id="typography">好的排版是无形的——它让读者专注于内容本身，而非排版设计。这是 Tufte 设计哲学的核心。</Sidenote> Edward Tufte 的设计哲学强调信息密度与视觉清晰度的平衡。

## 边注示例

在 Tufte 的设计中，边注（margin notes）是一个重要元素。<MarginNote id="MarginNote-example">边注不带编号，适合用于补充说明或引用来源。</MarginNote> 它们可以在不打断正文阅读流程的情况下，提供额外的上下文信息。

## 引用

<Blockquote author="Shawn O'Hare" source="Math is Fun" epigraph>
    This is an example of an epigraph with some math $ \mathbb N \subseteq \mathbb R $ to start the beginning of a section.
</Blockquote>

这个博客支持上面这种特殊的"epigraph"引用。当需要为每个大章节点缀一首诗，那么这种样式最合适不过。
另外博客中也支持行内引用，当正文中的某一句话需要表现出引用的效果，又不想打断整体的阅读体验时，只需要用 `<Cite>` 把要引用的文字框起来就可以了：<Cite>This is between cite tags and has math: $e^x $</Cite>

单独的引用块也可以，只要按照 markdown 的语法 `>` 就可以轻松实现。

For instance:

$$
\begin{aligned}
\mu(A) &= \iint_{I^2} \chi_A (x,y) \ d(x,y)
= \int_I \left( \int_I  \chi_A (x,y) \ dx\right) dy
= \int_I 0 \ dy= 0 \quad \text{and} \\
\mu(A) &=\iint_{I^2}  \chi_A (x,y) \ d(x,y)

\end{aligned}
